## Icon 图标

用于展示 icons 图标 。

## 代码演示

### 基本用法

在 ``template`` 中使用组件

```html
<press-icon type="back" size="30"></press-icon>
```

## API

### Icon Props

| 属性名 |  类型  | 默认值 |        说明        |
| :----: | :----: | :----: | :----------------: |
|  size  | Number |   24   |      图标大小      |
|  type  | String |   -    | 图标图案，参考示例 |
| color  | String |   -    |      图标颜色      |

### Icon Events

| 事件名 |        说明        | 返回值 |
| :----: | :----------------: | :----: |
| @click | 点击 Icon 触发事件 |   -    |
