# Popover

A type of popup layer, which is partial compared to Popup and can be used to host some tabs.

# Code Demo

# Basic usage

<div class="popover-wrap">
   <Press Popover
     :show="true"
     placement="bottom"
     custom-class="demo-nav-popover demo-nav-popover--bottom"
   >
     <div
       v-for="(item, index) of list"
       :key="index"
       class="demo-nav-item"
     >
       options {{ item }}
     </div>
   </PressPopover>
</div>

export default {
   data() {
     return {
       list: [1, 2, 3],
     }
   }
}

.popover-wrap {
   position: relative;
   width: 100%;
   height: 10px;

   .demo-nav-popover {
     top: 30px;
     bottom: auto;
     padding: 0;
     background: #09134e;
     &--top {
       left: 56px;
       right: auto;
     }
   }
   .demo-nav-item {
     display: flex;
     align-items: center;
     padding: 0 22px;
     height: 40px;
     color: #fff;
   }
}

property name type default value description
