# API

# Props

参数 说明 类型 默认值
total 页面总数 number -
current 当前页，从 1 开始 number -
tip-template 提示模板 string 第{{0}}页
safe-area-inset-bottom 是否留出底部安全距离 boolean true
auto-scroll 遇到边界自动滚动的页数 number 1

# Events

事件 说明 返回值
change 切换当前页 page
