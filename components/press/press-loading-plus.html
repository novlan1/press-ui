# LoadingPlus 加载中

加载中，分为图片模式和页面模式两种。

# 引入

import PressLoadingPlus from 'press-ui/press-loading-plus/press-loading-plus.vue';

export default {
  components: {
    PressLoadingPlus,
  }
}

# 基本用法

示例

<template>
  <press-loading-plus />
  <press-loading-plus loading-scenes="btn" />
  <press-loading-plus
    loading-scenes="page"
    loading-bg="#eee"
  />
</template>

<script>
import PressLoadingPlus from 'press-ui/press-loading-plus/press-loading-plus.vue';

export default {
  component: {
    PressLoadingPlus,
  },
};

</script>

# 实现原理

press-loading 采用的是绝对定位，left/right/bottom/top 均为0，所以要改变其高度，只需要修改最近的 position: relative的祖先元素（父元素）即可。
